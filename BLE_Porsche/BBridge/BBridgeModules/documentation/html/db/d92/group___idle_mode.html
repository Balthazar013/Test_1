uint8_t IdleMode_ExitMode ( void  )

It "exits" the idle mode. This function shall be called by the main worker controlling the "IDLE MODE" when it needs to switch between modes.

Returns
BB_SUCCESS: exit executed correctly
1: there was no previous command to set a different operating mode
