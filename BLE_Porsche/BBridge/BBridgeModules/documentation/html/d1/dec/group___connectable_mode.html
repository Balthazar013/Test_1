void ConnectableMode_Work ( void  )

State machine for the Connectable operating mode.

  • Wait initialization
  • Load configurations and set them in the bluetooth HW
  • Allows bluetooth stack to start broadcasting
  • Wait for connections
  • Validate peer upon connection
  • "Bridge" incomining bluetooth messages to the Panther
  • Sends received messages from the Panther to the validated connection
